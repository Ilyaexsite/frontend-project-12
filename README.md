## О проекте

React-приложение для чата с авторизацией и роутингом.

## Функциональность

- ✅ Роутинг (React Router DOM)
- ✅ Страница авторизации (Formik + Yup)
- ✅ 404 страница
- ✅ Деплой на Render

## Технологии

- React 18
- Vite
- TypeScript
- React Router DOM
- Formik + Yup для форм
- @hexlet/chat-server для бэкенда
